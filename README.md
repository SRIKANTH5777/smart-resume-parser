### ✨ Key Features

- Extracts:
  - ✅ Full Name
  - ✅ Email Address
  - ✅ Phone Number
  - ✅ Skills
  - ✅ Experience
  - ✅ Degree
  - ✅ College Name
  - ✅ Designation
  - ✅ Company Names
- Supports `.pdf` and `.docx` files
- CLI tool for batch parsing resumes
